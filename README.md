# LoadingSystem

Unity package for scene loading flow, published as UPM package `com.jis.loadingsystem`.

## UPM install (from GitHub)

Add this dependency to `Packages/manifest.json`:

```json
{
  "dependencies": {
    "com.jis.loadingsystem": "https://github.com/KunNguyen/LoadingSystem.git?path=Assets/com.jis.loadingsystems"
  }
}
```

## Package dependency

`com.jis.loadingsystem` depends on UniTask via Git URL:

```json
"com.cysharp.unitask": "https://github.com/Cysharp/UniTask.git?path=src/UniTask/Assets/Plugins/UniTask"
```
